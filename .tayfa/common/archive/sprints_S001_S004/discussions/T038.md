# Task discussion T038: Game Designer: UX-ревью карты мира и потока игры

## [2026-02-26 15:02] boss (author)

### Task description

Ревью нового пользовательского опыта после добавления карты мира.

Через Agent Bridge (см. prompt.md) пройди полный flow:
1. Главное меню -> Карта мира -> Уровень -> Завершение -> Возврат на карту

Оцени:
1. КАРТА МИРА:
   - Понятно ли где ты, куда можешь идти, что уже прошёл?
   - Есть ли ощущение 'мира' и 'исследования'?
   - Работает ли нелинейность? (можно ли выбрать путь?)
   - Визуальные состояния залов читаются? (LOCKED vs AVAILABLE vs COMPLETED)

2. ПЕРЕХОДЫ:
   - Плавный ли переход меню -> карта -> уровень -> карта?
   - Не теряется ли контекст при возврате на карту?
   - Понятно ли что уровень завершён?

3. ЭХО-ПОДСКАЗКИ:
   - Не раздражают ли? Достаточно ли мягкие?
   - Помогают ли реально решить головоломку?
   - Прогрессия шёпот -> голос -> видение ощущается?

4. ОБЩЕЕ ВПЕЧАТЛЕНИЕ:
   - Хочется ли исследовать дальше?
   - Есть ли ощущение прогресса и достижения?
   - Что бы ты изменил в первую очередь?

Доставь: подробный UX-отчёт с оценками 1-10 по категориям и конкретными рекомендациями.
