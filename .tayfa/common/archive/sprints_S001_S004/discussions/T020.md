### Task description

Перевести ВСЕ пользовательские тексты на русский. Это включает:

1. **12 JSON-файлов уровней** (data/levels/act1/level_01..12.json): title, subtitle, automorphism name/description, hint text.
2. **level_scene.gd**: все hardcoded строки — кнопки (RESET, TEST PATTERN, COMBINE KEYS, NEXT LEVEL), сообщения туториала, статус-сообщения, подсказки, описания групп, заметки 'что вы узнали', панель генераторов, панель завершения.

Примеры перевода:
- 'The Triangle Vault' → 'Треугольный зал'
- 'Three crystals, three secrets' → 'Три кристалла, три секрета'
- 'Identity' → 'Тождество'
- 'Rotation 120°' → 'Поворот на 120°'
- 'Everything stays in place' → 'Всё остаётся на месте'
- 'RESET' → 'СБРОС'
- 'TEST PATTERN' → 'ПРОВЕРИТЬ УЗОР'
- 'Keys: 0 / 3' → 'Ключи: 0 / 3'
- 'Vault Opened!' → 'Зал открыт!'

Важно: математические термины переводить на понятный русский, не научным стилем. Это игра, не учебник.

МУСТ: после перевода запустить игру через Agent Bridge и убедиться, что все тексты отображаются корректно (нет обрезанных строк, кодировка OK).
