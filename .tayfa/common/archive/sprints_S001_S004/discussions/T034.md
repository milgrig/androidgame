# Task discussion T034: Система Эхо-подсказок (Echo Hints)

## [2026-02-26 15:02] boss (author)

### Task description

Реализовать трёхуровневую систему подсказок из redesign.md (пункт 9).

Три уровня подсказок:
1. Эхо-шёпот (мягкий намёк): появляется через 60 сек бездействия. Пример: 'Этот кристалл похож на другой...'
2. Эхо-голос (конкретная подсказка): появляется через 120 сек. Пример: 'Попробуй поменять кристаллы 1 и 3'
3. Эхо-видение (визуальная демонстрация): появляется через 180 сек. Подсветка нужных кристаллов пульсацией.

Реализация:
1. Создать src/game/echo_hint_system.gd:
   - Timer-based триггеры (idle_time tracking)
   - 3 уровня с прогрессивным раскрытием
   - Данные подсказок берутся из level JSON (поле hints уже существует)
   - Сигналы: hint_shown(level, text), hint_dismissed

2. Создать визуальные компоненты:
   - Текстовая подсказка (Label с fade-in анимацией)  
   - Подсветка кристаллов (glow-эффект на целевых нодах)

3. Интегрировать в LevelScene: подключить к idle detection
4. Обновить level JSON файлы: расширить hints до 3 уровней где нужно

MUST прочитать: redesign.md (пункт 9), существующие hints в level_01.json..level_12.json
