# Task discussion T022: Game Designer: UX-ревью всех 12 уровней через Agent Bridge

## [2026-02-26 14:55] boss (author)

### Task description

Запустить игру через Agent Bridge и пройти все 12 уровней как скептичный игрок-нематематик. Оценить:

1. **Прогрессия сложности (1-10)**: плавная ли кривая обучения от уровня 1 к 12?
2. **Aha-момент (1-10)**: когда игрок обнаруживает симметрию — это приятно?
3. **Понятность подсказок (1-10)**: помогают ли hints понять, что делать?
4. **Разнообразие (1-10)**: не надоедает ли к уровню 8?
5. **Обучающий эффект (1-10)**: научится ли игрок ВИДЕТЬ симметрии к уровню 12?
6. **Тексты на русском (1-10)**: звучат ли естественно? Нет ли корявых переводов?

Для каждого уровня: краткий вердикт + конкретные проблемы + предложения.

Особое внимание:
- Уровни 7-9 (неожиданные симметрии): понимает ли игрок, почему нерегулярный граф имеет симметрию?
- Уровни 10-12 (генераторы): понятна ли концепция 'один ключ порождает все'?
- Переход от Z3 к D4: не слишком ли резкий скачок?

Используй Agent Bridge (см. свой prompt.md для инструкций).
