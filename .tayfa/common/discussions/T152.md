# Task discussion T152: Layer 2 — выбор задания в панели слева

## [2026-03-03 17:30] boss (author)

### Task description

Панель зеркальных пар (mirror_pairs_panel) слева показывает список заданий — для каждого ключа найти обратный. Игрок должен мочь кликнуть на ЗАДАНИЕ чтобы выбрать, какому ключу ищет пару.

СЕЙЧАС: _active_slot = первый неспаренный (автоматически). Игрок не может выбрать.

НУЖНО:
- Каждый неспаренный слот — кликабельный
- Клик на неспаренный слот → он становится активным (_active_slot = idx)
- Активный слот подсвечен (как сейчас: стиль "active", стрелка "<-")
- Клик на уже спаренный слот → ничего
- Клик на уже активный слот → ничего (или сброс)

ВИЗУАЛЬНО:
- Неспаренные слоты имеют cursor pointer (кликабельны)
- Активный слот: яркая рамка + "<-"
- Неактивные неспаренные: обычная рамка, но с hover эффектом

ФАЙЛ: src/ui/mirror_pairs_panel.gd
- Добавить обработчик клика на слоте
- _on_slot_clicked(idx) → если не спарен → set_active_slot(idx)
- Сигнал: active_slot_changed(idx) — чтобы layer_mode_controller знал

### Acceptance criteria

- [ ] Клик на неспаренный слот → он становится активным
- [ ] Активный слот подсвечен визуально
- [ ] Клик на спаренный → ничего
- [ ] Игрок может выбирать задания в произвольном порядке
- [ ] При нахождении пары → слот блокируется, автоматически выбирается следующий неспаренный
