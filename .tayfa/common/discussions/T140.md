# Task discussion T140: QA — визуальное тестирование всех слоёв с кластерами

## [2026-03-03 12:54] boss (author)

### Task description

Визуальное тестирование Room Cluster Overlay на всех 5 слоях.

ПРОВЕРИТЬ:
1. **Layer 1**: Никаких кластеров (не должно ничего сломаться)
2. **Layer 2**: Зелёные capsule вокруг зеркальных пар, жёлтые halo для самозеркальных
3. **Layer 3**: Золотой кластер вокруг выбранной подгруппы
4. **Layer 4**: Красный кластер вокруг выбранной подгруппы
5. **Layer 5**: Разноцветные кластеры вокруг классов смежности

EDGE CASES:
- Группа Z2 (2 комнаты): кластер вокруг 1 комнаты (halo)
- Группа Z3 (3 комнаты): подгруппа из 1 или все 3
- Группа D4 (8 комнат): подгруппы разных размеров (2, 4), факторгруппы
- Группа S4 (24 комнаты): большие группы, далёкие комнаты, пунктирные дуги

ГИБРИД:
- Проверить что далёкие комнаты одной группы → отдельные обводки + пунктирные дуги
- Проверить что близкие комнаты → общий пузырь
- Проверить что номера комнат читаемы под контурами

### Acceptance criteria

- [ ] Все 5 слоёв работают корректно
- [ ] Кластеры визуально красивые и не мешают геймплею
- [ ] Edge cases протестированы
- [ ] Гибридный подход (близкие/далёкие) работает правильно
- [ ] Нет регрессий в существующей функциональности
