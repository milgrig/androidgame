### Task description

Написать Python unit-тесты для алгоритмов Room Cluster Overlay.

ТЕСТЫ НУЖНЫ ДЛЯ:
1. **Convex Hull** — Andrew's monotone chain:
   - Треугольник (3 точки) → 3 вершины
   - Квадрат (4 точки) → 4 вершины
   - Точки внутри hull не включаются
   - Коллинеарные точки
   - 1 и 2 точки (degenerate cases)

2. **Subclustering** (разбиение на подкластеры по расстоянию):
   - Все рядом → 1 подкластер
   - Все далеко → N подкластеров (по одному на комнату)
   - Частичное: 2 рядом + 1 далеко → 2 подкластера
   - Цепочка: A-B рядом, B-C рядом → все в одном (транзитивность)

3. **API** set_room_clusters / clear_room_clusters:
   - Установка кластеров обновляет данные
   - Очистка сбрасывает данные
   - Пустой массив = нет кластеров

ФАЙЛ: tests/fast/unit/test_room_cluster.py (новый)

NOTE: Алгоритмы convex hull и subclustering — чистая математика, тестируемая без Godot. Можно извлечь в отдельный utility module или тестировать порт на Python.
