# Task discussion T143: Клик на комнату = телепорт кристаллов

## [2026-03-03 15:44] boss (author)

### Task description

Реализовать функционал: клик на комнату на room_map → кристаллы перемещаются в позиции этой комнаты.

СЕЙЧАС: room_clicked сигнал существует, хит-тест работает, но обработчик в level_scene.gd — пустой (pass).

НУЖНО:
```gdscript
func _on_room_map_clicked(room_idx: int) -> void:
    if room_idx == _room_state.current_room:
        return  # уже в этой комнате
    if not _room_state.is_discovered(room_idx):
        return  # комната не открыта

    # 1. Вычислить перестановку для перехода в эту комнату
    # 2. Обновить arrangement кристаллов
    # 3. Анимировать кристаллы (0.5с) на новые позиции
    # 4. Обновить room_state.current_room
    # 5. Обновить подсветку текущей комнаты на карте
```

СУЩЕСТВУЮЩИЕ ИНСТРУМЕНТЫ:
- room_state.apply_key(key_idx) — применяет ключ
- swap_manager.apply_arrangement_to_crystals() — анимирует кристаллы
- ShuffleManager.build_positions_map() — вычисляет позиции

ВАЖНО: Каждая комната = конкретная перестановка. Нужно:
1. Найти перестановку target_room из room_state (это perm_ids[room_idx])
2. Вычислить обратную от текущей перестановки
3. Применить composition: inverse(current) * target → получаем переход
4. Или проще: напрямую установить arrangement для target_room

ФАЙЛЫ:
- src/game/level_scene.gd — обработчик _on_room_map_clicked (строка 471)
- src/game/swap_manager.gd — анимация кристаллов
- src/game/room_state.gd — текущая комната, перестановки
