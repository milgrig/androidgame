### Task description

СЕЙЧАС: На Layer 2 зеркальные пары показываются ТОЛЬКО на панели слева. На room_map — никакой визуализации.

НУЖНО: При нахождении каждой зеркальной пары — обновлять кластеры на карте. Все найденные пары отображаются одновременно.

ЛОГИКА:
```gdscript
var clusters: Array = []
for pair in found_pairs:
    if pair.is_self_inverse:
        # Самозеркальный — 1 комната, жёлтый halo
        clusters.append({
            "rooms": [pair.key_room_idx],
            "color": Color(1.0, 0.85, 0.3, 0.9),  # L2_SELF_COLOR (жёлтый)
            "label": "",
        })
    else:
        # Пара — 2 комнаты, зелёный capsule
        clusters.append({
            "rooms": [pair.key_room_idx, pair.inv_room_idx],
            "color": Color(0.2, 0.85, 0.4, 1.0),  # L2_GREEN
            "label": "",
        })
room_map.set_room_clusters(clusters)
```

Вызывать обновление каждый раз когда пара найдена (из _on_pair_matched или аналогичного обработчика).
