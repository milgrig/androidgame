# Task discussion T138: Layer 5 — классы смежности через Room Clusters

## [2026-03-03 12:54] boss (author)

### Task description

Добавить кластерные контуры поверх существующей coset coloring на Layer 5.

СЕЙЧАС: set_coset_coloring() перекрашивает комнаты по цветам классов. Есть dashed внутренние рёбра и curved внешние. Нет визуального контура-пузыря вокруг класса.

НУЖНО: Помимо перекраски — добавить кластерные контуры:
```gdscript
var clusters: Array = []
for ci in range(cosets.size()):
    var coset = cosets[ci]
    var rooms = sym_ids_to_room_indices(coset["elements"])
    clusters.append({
        "rooms": rooms,
        "color": COSET_COLORS[ci % COSET_COLORS.size()],
        "label": "%sN" % coset["representative"],
        "show_internal_edges": false,  # уже есть coset edges
        "fill_alpha": 0.10,
    })
room_map.set_room_clusters(clusters)
```

ВАЖНО: set_coset_coloring() ОСТАЁТСЯ для перекраски комнат внутри. Кластеры добавляют КОНТУРЫ поверх. Это два независимых слоя визуализации.

Гибридный подход особенно важен здесь: в факторгруппах все комнаты должны поделиться на классы. Большие hull'ы перекроются — поэтому далёкие комнаты одного класса показываются отдельными обводками + дугами.

ФАЙЛ: src/game/layer_mode_controller.gd — секция Layer 5, вызовы set_coset_coloring.
