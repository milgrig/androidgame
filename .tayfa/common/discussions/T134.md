# Task discussion T134: Core — Room Cluster Overlay в room_map_panel.gd

## [2026-03-03 12:54] boss (author)

### Task description

Добавить универсальную систему визуального объединения комнат на room_map.

КЛЮЧЕВОЙ ДИЗАЙН: ГИБРИДНЫЙ ПОДХОД (близкие vs далёкие комнаты)
- Комнаты в одной группе, которые РЯДОМ → один общий пузырь (convex hull / capsule)
- Комнаты в одной группе, которые ДАЛЕКО → каждая со своей обводкой + пунктирные дуги между ними
- Порог: CLUSTER_MERGE_DIST = node_size * 3.5 (~100px для sz=28)
- Union-find для разбиения на подкластеры

ФОРМЫ ПОДКЛАСТЕРОВ:
- 3+ комнат в подкластере: Rounded convex hull (Andrew's monotone chain + расширение + скругление)
- 2 комнаты: Capsule (скруглённый прямоугольник)
- 1 комната: Rounded glow halo (увеличенная рамка)

РИСОВАНИЕ:
- Полупрозрачная заливка (fill_alpha ~0.08)
- Цветной контур (alpha 0.45, width 1.5px)
- 2 glow-слоя снаружи (убывающая alpha)
- Пунктирные безье-дуги между далёкими подкластерами (alpha 0.3)
- Опциональная метка в центроиде ("H", "N", "gN")

API:
```gdscript
func set_room_clusters(clusters: Array) -> void
# clusters = [{rooms: Array[int], color: Color, label: String, show_internal_edges: bool, fill_alpha: float}]

func clear_room_clusters() -> void
func sym_ids_to_room_indices(sym_ids: Array) -> Array
```

ПОРЯДОК _draw():
1. Кластеры (ниже комнат, выше фона)
2. Legacy fallback (если кластеры не установлены)
3. Fading edges, Key preview
4. Room nodes (всегда сверху)

ФАЙЛ: src/game/room_map_panel.gd (~1046 строк)
