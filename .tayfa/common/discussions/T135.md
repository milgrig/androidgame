### Task description

Заменить вызовы highlight_subgroup() на set_room_clusters() в layer_mode_controller.gd для Layer 3.

ЧТО СЕЙЧАС: При выборе подгруппы (брелка) → highlight_subgroup(sym_ids) → золотые glow-кольца + золотые curved рёбра. Нет контура вокруг группы.

ЧТО НУЖНО: При выборе подгруппы → set_room_clusters() с одним кластером:
```gdscript
room_map.set_room_clusters([{
    "rooms": sym_ids_to_room_indices(elements),
    "color": Color(0.95, 0.80, 0.20, 1.0),  # L3_GOLD
    "label": "H",
    "show_internal_edges": true,
    "fill_alpha": 0.08,
}])
```

При снятии выбора → clear_room_clusters().
