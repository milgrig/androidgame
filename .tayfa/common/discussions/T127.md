# Task discussion T127: Шаг 2: Игрок определяет тип факторгруппы (выбор из вариантов)

## [2026-03-02 11:51] boss (author)

### Task description

КОНТЕКСТ: После того как игрок правильно собрал классы смежности (Шаг 1), нужно определить тип получившейся факторгруппы.

МЕХАНИКА:
1. После сборки классов показывается склеенный граф факторгруппы (анимация склеивания).
2. Игрок видит маленький граф G/N с операциями между классами.
3. Появляется вопрос: Какая это группа?
4. Варианты ответа (3-4 кнопки): например [Z2] [Z3] [Z4] [V4].
5. Варианты генерируются: правильный ответ + 2-3 отвлекающих того же порядка или близких.
6. Правильный ответ = факторгруппа засчитана, зелёная анимация.
7. Неправильный = красная вспышка, можно попробовать снова.
8. Когда ВСЕ факторгруппы уровня определены — Слой 5 пройден.

ГЕНЕРАЦИЯ ВАРИАНТОВ:
- Правильный: quotient_type из JSON
- Отвлекающие: группы того же порядка (Z4 vs V4), или ±1 порядок (Z2 vs Z3)
- Пул: Z2, Z3, Z4, Z5, Z6, V4, Z2xZ2, Z2xZ3, S3, D3, D4, A4
